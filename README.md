# SKKUEasyPortal
성균관대 학생을 위한 Easy - 포털 (Web Page ver)

학생들의 일정관리, 강의 수강, 학교 공지 확인(학교(과) 공지사항, gls, I-campus 등) 등 대학생활에 필요한 내용들이 너무 분산되어 있다는 생각이 들었다. 

이 내용들에 대한 접근성을 높이기 위해, 학교 생활에 필요한 내용을 쉽게 관리할 수 있는 web application을 제작해보고자 했다.

**< 제공 기능 >**
1. **[궁금해](#--궁금해)** : 학교 생활에 필요한 정보 (졸업 요건 등) 정리
2. **[공부해](#--공부해)** : 공부를 돕는 타이머
3. **[암기해](#--암기해)** : 과목별 파일 저장

### 1. 타이머 
text 창에 과목 이름을 넣고 enter 키를 누르면, 노란색으로 과목 이름이 설정된다. 그리고 플레이버튼을 누르면 공부하고 있는 시간을 잴 수 있다.
다시 일시정지를 누르면 시간이 멈추며, 내가 총 공부한 시간을 확인할 수 있다.

사용자가 로그인을 하면, 공부하고 있는 과목 이름을 다시 설정하지 않아도 다음 로그인을 하였을 때, 그대로 저장되어 있다.
새로 과목이름을 바꾸고 싶다면, text창에 다시 입력하면 된다.

### 2. D-day 리스트
시험을 포함한 학사 중요한 일정이 D-day로 표기되어 있어서 확인할 수 있다.
매일 자동으로 D-day 가 계산되어 표시된다.

### 3. Useful Link for Studying
공부할 때 도움이 될만한 사이트와 링크가 소개되어있다.
링크를 누르면 새창에서 해당 링크로 연결된다.
